# Initial Set Up
```bash
brew install awscli hyper python terraform
```
# Installations
## Homebrew
    ```
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```
## fish
    ```
    brew install fish
    ```

## Oh-My-Posh
    ```
    brew tap jandedobbeleer/oh-my-posh
    brew install oh-my-posh
    ```
## AWS CLI
    ```
    brew install awscli
    ```
